-----------------------------------------------------------------------------------------
Le .zip 'node_modules' contient trop de dossiers pour être uploader. 
Il faut le dézipper avant de pouvoir lancer le projet.
-----------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------
La base de données s'appelle '420Node' et les différentes collections sont situées dans 
dossier 'database'.
Le nom des collections ne doit pas être changé lors de l'importation.
-----------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------
Pour lancer le serveur depuis un terminal le raccourci est 'devStart' ou 'start'
-> npm run devStart
-> npm run start
